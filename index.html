Introducing Dotfiles

Dotfiles aggregates a collection of standalone configuration files (dotfiles) that can be used to customize your development environment across numerous computers and operating systems.

Version: v0.2.450

Download View on GitHub

Easy to get started.

All you have to do is to launch the dotfiles installer, and it will do the rest for you. There is a minimal amount of rules to learn.

Lightweight

Dotfiles comes packed with powerful features, including Bash Shell preferences, Homebrew essentials, macOS Preferences; all in just 115 KB

Simpler Structure

Each directory only contained a handful of files and rules.
