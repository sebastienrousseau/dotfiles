# Dotfiles (v0.2.468)

Simply designed to fit your shell life 🐚

![Dotfiles banner][banner]

## 🆆🅶🅴🆃 🅰🅻🅸🅰🆂🅴🆂

This code provides a set of aliases for the `wget` command.

- `wg` wget.
- `wgc` wget with continue.
- `wge` wget with robots=off.
- `wget` wget with continue.
