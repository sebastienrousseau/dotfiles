# Dotfiles (v0.2.468)

Simply designed to fit your shell life 🐚

![Dotfiles banner][banner]

## 🅼🅰🅺🅴 🅰🅻🅸🅰🆂🅴🆂

This code provides a set of aliases for the `make` command.

- `mk` - Make.
- `mkc` - Make clean.
- `mkd` - Make doc.
- `mkf` - Make format.
- `mkh` - Make help.
- `mki` - Make install.
- `mka` - Make all.
- `mkr` - Make run.
- `mkt` - Make test.
