This code provides a set of aliases for the `make` command.
- `mk` - Make.
- `mkc` - Make clean.
- `mkd` - Make doc.
- `mkf` - Make format.
- `mkh` - Make help.
- `mki` - Make install.
- `mka` - Make all.
- `mkr` - Make run.
- `mkt` - Make test.
