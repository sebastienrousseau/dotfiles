# Python Aliases

Manage Python aliases. Part of the **Universal Dotfiles** configuration.

![Dotfiles banner][banner]

## Description

These aliases are defined in `python.aliases.sh` and are automatically loaded by `chezmoi`.

## Aliases

- `py` - python
- `pip` - pip3
- `v` - virtualenv
- `uv` - uv package manager
- `uvpi` - `uv pip install`
