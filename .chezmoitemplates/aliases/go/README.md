# Go Aliases

Manage Go language aliases. Part of the **Universal Dotfiles** configuration.

![Dotfiles banner][banner]

## Description

These aliases are defined in `go.aliases.sh` and are automatically loaded by `chezmoi`.

## Aliases

- `go` - Go shortcut
- `gor` - `go run`
- `gob` - `go build`
- `got` - `go test`
- `gomt` - `go mod tidy`
- `goget` - `go get`
