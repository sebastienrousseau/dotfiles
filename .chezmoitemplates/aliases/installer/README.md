# Installer Aliases

Tools for bootstrapping and remote deployment.

## Commands

| Alias | Description |
|-------|-------------|
| `dot-install` | Run the `install.sh` bootstrap script locally. |
| `dot-teleport` | Teleport your config to a remote server (`dot-teleport user@host`). |
