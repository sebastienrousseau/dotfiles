Introducing Dotfiles

Dotfiles are a set of macOS, Linux and Windows configuration files. They are aimed at making your life easier by providing a set of scripts and configuration files with new ways to get things done.

What are Dotfiles?

Dotfiles aggregates a collection of standalone configuration files (dotfiles) combined into a shell directory that can be used to customize your development environment across numerous computers and operating systems (macOS, Windows, Linux).

The Dotfiles provides modular configuration files (aliases, functions and paths) built for speed, higher performance, with the aim of helping you have an easy and centralized way to configure your environment and applications.

Easy to get started.

All you have to do is to launch the dotfiles installer, and it will do the rest for you. There is a minimal amount of rules to learn.

Lightweight

Dotfiles comes packed with powerful features, including Bash Shell preferences, Homebrew essentials, macOS Preferences; all in just 545 kB

Simpler Structure

Each directory only contained a handful of files and rules.
